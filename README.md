# 项目说明
- 该项目是基于immortalwrt-master进行的定制
- 定制内容不一定符合所有人的需求，可自行 fork 后修改
- 适配机型：WXY-OES、WXY-OEC/OECT
- 插件说明：X86仅打包PassWall2、HomeProxy，纯科学旁路由，不带docker
- 编译时间：每周五 13:00 后 开始自动编译，编译时间大概2-3小时
